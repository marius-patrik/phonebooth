# Phonebooth Workspace 
Setup  This workspace contains both the Phonebooth frontend and backend projects configured to run together in VS Code.  ## Quick Setup (New Machine)  1. **Clone both repositories:**    ```powershell    mkdir phonebooth    cd phonebooth    git clone https://github.com/pastiiiiiiik/phonebooth.git phonebooth-frontend    git clone https://github.com/pastiiiiiiik/phoneserver.git phonebooth-backend Download the workspace file:  Download phonebooth.code-workspace from this repo Place it in the phonebooth/ folder Open in VS Code:  code phonebooth.code-workspace Install dependencies: Both dev servers will auto-start. If they fail, install dependencies first:  cd phonebooth-frontend npm install cd ../phonebooth-backend npm install Features Auto-start dev servers: Both frontend (port 3000) and backend (port 8080) start automatically when you open the workspace Split terminals: Servers run in dedicated terminal panels Separate Git repos: Each project has its own Git repository and can be committed/pushed independently TypeScript IntelliSense: Properly configured for both projects Biome formatting: Auto-format on save Manual Tasks If auto-start is disabled, run:  Frontend: Dev Server task for the frontend Backend: Dev Server task for the backend Project Structure phonebooth/ ├── phonebooth.code-workspace    # VS Code workspace configuration ├── phonebooth-frontend/         # React + Rsbuild app └── phonebooth-backend/          # Express + SQLite API GitHub Repositories Frontend: https://github.com/pastiiiiiiik/phonebooth Backend: https://github.com/pastiiiiiiik/phoneserver
